Year 3 Applied Electricity - Digital Electronics

Digital Electronics Content

1. Binary Numbers

Binary numbers are the foundation of digital electronics, using only two digits: 0 and 1. This system is used because electronic circuits have two stable states: ON (1) and OFF (0).

1.1 Binary Number System

  • Base-2 number system
  • Digits: 0 and 1 (called bits)
  • Each position represents a power of 2

1.2 Binary to Decimal Conversion

To convert binary to decimal, multiply each bit by its positional value and sum the results.

Example: 1011 (binary) = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11 (decimal)

1.3 Decimal to Binary Conversion

To convert decimal to binary, divide the number by 2 repeatedly and read the remainders from bottom to top.

Example: 13 (decimal) to binary:

13 ÷ 2 = 6 remainder 1
6 ÷ 2 = 3 remainder 0
3 ÷ 2 = 1 remainder 1
1 ÷ 2 = 0 remainder 1

Reading from bottom to top: 13 (decimal) = 1101 (binary)

2. Logic Gates

Logic gates are the basic building blocks of digital circuits. They perform Boolean operations on binary inputs to produce binary outputs.

2.1 Basic Logic Gates

  • AND Gate: Output is 1 only if all inputs are 1
  • OR Gate: Output is 1 if at least one input is 1
  • NOT Gate (Inverter): Output is the opposite of the input
  • NAND Gate: Combination of AND followed by NOT
  • NOR Gate: Combination of OR followed by NOT
  • XOR Gate: Output is 1 if inputs are different
  • XNOR Gate: Output is 1 if inputs are the same

2.2 Truth Tables

Truth tables show all possible input combinations and their corresponding outputs for a logic gate.

Example: AND Gate Truth Table

2.3 Boolean Algebra

Boolean algebra is used to analyze and simplify digital circuits. Key laws include:

  • Commutative Law: A + B = B + A, A · B = B · A
  • Associative Law: (A + B) + C = A + (B + C), (A · B) · C = A · (B · C)
  • Distributive Law: A · (B + C) = (A · B) + (A · C)
  • Identity Law: A + 0 = A, A · 1 = A
  • Complement Law: A + A' = 1, A · A' = 0

Work Solutions

Problem 1: Binary to Decimal Conversion

Convert the binary number 10110 to decimal.

Solution:

10110 (binary) = 1×2⁴ + 0×2³ + 1×2² + 1×2¹ + 0×2⁰

= 16 + 0 + 4 + 2 + 0

= 22 (decimal)

Problem 2: Decimal to Binary Conversion

Convert the decimal number 45 to binary.

Solution:

45 ÷ 2 = 22 remainder 1
22 ÷ 2 = 11 remainder 0
11 ÷ 2 = 5  remainder 1
5 ÷ 2 = 2   remainder 1
2 ÷ 2 = 1   remainder 0
1 ÷ 2 = 0   remainder 1

Reading from bottom to top: 45 (decimal) = 101101 (binary)

Problem 3: Logic Gate Output

Determine the output of an AND gate with inputs A = 1 and B = 0.

Solution:

For an AND gate, the output is 1 only if all inputs are 1.

Since A = 1 and B = 0, the output will be 0.

Problem 4: Boolean Algebra Simplification

Simplify the Boolean expression: A · B + A · (B + C) + (B + C)

Solution:

Step 1: A · B + A · (B + C) + (B + C)

Step 2: A · B + A · B + A · C + B + C (Distributive Law)

Step 3: (A · B + A · B) + (A · C + B + C)

Step 4: A · B + (A · C + B + C) (Idempotent Law: X + X = X)

Step 5: A · B + B + C (Absorption Law: X + X · Y = X)

Step 6: B + C (Absorption Law again)

Therefore, the simplified expression is B + C

Problem 5: Combinational Logic Circuit

Design a logic circuit that implements the function F = A · B + C

Solution:

1. Use an AND gate for A · B

2. Use an OR gate to combine the output of the AND gate with C

The circuit would have two inputs (A and B) going into an AND gate, and the output of the AND gate along with input C going into an OR gate. The output of the OR gate gives the function F.
